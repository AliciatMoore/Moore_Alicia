Tags

Scope

The scope is defined by the opening and closing tags.

Head

"< head >< / head >"
Includes information for the browser and is not seen by the user. Includes keywords for search engines, browser info, title of the website etc.

Body

"< body >< / body >"
Includes all of the actual site content, such as paragraphs, images, audio, animations, backgrounds, etc.
--Paragraphs < p >< / p >
Spaces, tabs, and return characters are not translated onto the visible body elements of the site. The paragraph tag allows for telling the browser which text is going to be in paragraph form.
--Inline text tags < br / >
Used to add a single space. One tag allows for one single space. ex: double space is < br/ >< br/ >
--Horizontal Rule < hr />
Adds a horizontal line.
--< strong > tag
Bolds the enclosed text.
--< em > tag
Italicizes the enclosed text

Adding links

: < a >< / a > "anchor" tag allows for linking to:
--other sites: < a href = "http://www.sduhsd.net/subsites/my-sduhsd-net/">History< / a >
--other pages on your own site:< a href = "page2.html">Go to page two < / a >
--specific places in a page( page navigation ): < a href = "#history">History< / a > and < h3>< a name = "history">< / a>History< / h3>
